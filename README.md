Atualização do App Campo para conectar com a API Google Apps Script da Central Açaí Control.

- `index.html` — app atualizado.
- `service-worker.js` — quebra de cache V1.5.
- manter o `manifest.json` atual.
- manter a pasta `icons/` atual.

## Teste no celular

1. Abrir o app pelo GitHub Pages.
2. Entrar na aba **Fazenda**.
3. Colar a URL do App da Web do Apps Script.
4. Informar a chave da fazenda teste.
5. Clicar em **Validar chave online**.
6. Fazer uma venda teste.
7. Entrar em **Pendências** e clicar em **Sincronizar agora**.
8. Conferir a aba `REGISTROS_CAMPO` na planilha Google.
